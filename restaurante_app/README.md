## Descripción
Este proyecto es un sistema básico desarrollado en Python usando Programación Orientada a Objetos. Permite gestionar productos y clientes dentro de un restaurante.

## Estructura del proyecto

- modelos:" contiene las clases Producto y Cliente
- servicios: contiene la lógica principal del sistema (Restaurante)
- main.py: archivo principal de ejecución

## Importancia de la modularidad
La modularidad permite organizar el código en diferentes archivos y responsabilidades, facilitando el mantenimiento, reutilización y comprensión del sistema.

## Reflexión
Separar responsabilidades hace que el sistema sea más claro, escalable y fácil de modificar en el futuro.
